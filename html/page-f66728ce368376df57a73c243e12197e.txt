// cookie handler functions
function createCookie(name,value,days) {
	if (days) {
		var date = new Date();
		date.setTime(date.getTime()+(days*24*60*60*1000));
		var expires = "; expires="+date.toGMTString();
	}
	else var expires = "";
	document.cookie = name+"="+value+expires+"; path=/";
}
function readCookie(name) {
	var nameEQ = name + "=";
	var ca = document.cookie.split(';');
	for(var i=0;i < ca.length;i++) {
		var c = ca[i];
		while (c.charAt(0)==' ') c = c.substring(1,c.length);
		if (c.indexOf(nameEQ) == 0) return c.substring(nameEQ.length,c.length);
	}
	return null;
}

// the logic upon pageload
var wherefrom = "";
wherefrom = document.referrer;

if(wherefrom.indexOf('facebook.com') > 0) {
	wherefrom = "facebook";
} else if(wherefrom.indexOf('twitter.com') > 0) {
	wherefrom = "twitter";
} else if(wherefrom.indexOf('blog.truecar.com') > 0) {
	wherefrom = "blog_truecar";
} else {
	wherefrom = "null";
}

// don't overwrite original referrer
if(readCookie('blogrfr')) {
	// it's already set; don't do a thing!
} else {
	createCookie('blogrfr',wherefrom,.05);
}

<p>Reporters who endured <a title="get specs and pricing on a variety of Volkswagen models" href="http://www.truecar.com/prices-new/volkswagen/" >Volkswagen</a>'s goofy interpretive/modern/hiphop dance routines at the North American International Auto Show were rewarded with a sneak peek at Volkswagen's first hybrid sedan.<span id="more-19144"></span></p>
<h2><strong>2013 Volkswagen Jetta Hybrid Powertrain</strong></h2>
<p>Powered by a 150-horsepower/1.4-liter turbocharged four-cylinder engine mated to a 20-kW electric motor, the new hybrid Jetta sedan promises an average combined fuel economy of about 45 miles per gallon, and acceleration from 0 to 60 mph in less than 9 seconds. Meanwhile, the maximum horsepower of this powertrain surpasses what's produced by the 170-horsepower/2.5-liter five-cylinder engine found in the conventionally powered <a title="get specs and pricing on the Volkswagen Jetta" href="http://www.truecar.com/prices-new/volkswagen/jetta-sedan-pricing/" >Jetta sedan</a>.</p>
<p><a title="2013-Volkswagen-Jetta-Hybrid" href="http://blog.truecar.com/2012/01/19/volkswagen-reveals-2013-jetta-hybrid-at-2012-north-american-international-auto-show/2013-volkswagen-jetta-hybrid1/"  rel="attachment wp-att-19154" target="_blank">
		<span class="pibfi_pinterest">
		<img class="alignright size-full wp-image-19154" style="border-style: initial; border-color: initial; border-image: initial; border-width: 0px; margin: 5px;" title="2013-Volkswagen-Jetta-Hybrid1" src="http://tcblog.truecar.com/wp-content/uploads/2012/01/2013-Volkswagen-Jetta-Hybrid11.jpg" alt="2013-Volkswagen-Jetta-Hybrid" width="400" height="266" />
			<span class="xc_pin" onclick="pin_this(event, 'http://pinterest.com/pin/create/button/?url=http://blog.truecar.com/2012/01/19/volkswagen-reveals-2013-jetta-hybrid-at-2012-north-american-international-auto-show/&media=http://tcblog.truecar.com/wp-content/uploads/2012/01/2013-Volkswagen-Jetta-Hybrid11.jpg&description=Volkswagen Reveals 2013 Jetta Hybrid at 2012 North American International Auto Show')">
			</span>
		</span></a>Thanks to its 220-volt, 1.1kWh lithium-ion battery pack, the 2013 Jetta hybrid can be driven in electric-only mode at speeds up to about 45 mph for a distance of about 1.3 miles, depending on terrain and operating conditions.</p>
<p>Much like its big brother, the <a title="get specs and pricing on the Volkswagen Touareg SUV" href="http://www.truecar.com/prices-new/volkswagen/touareg-pricing/" >Volkswagen Touareg</a> hybrid SUV, the Jetta hybrid compact sedan boasts a decoupling clutch that fully disengages the gasoline engine from the drivetrain at higher engine speeds (up to 84 mph) to reduce engine drag and allow the vehicle to "sail" or coast for longer distances without consuming any fuel. The engine also decouples whenever the driver releases the gas pedal to brake, at which time regenerative braking transfers kinetic energy to the batteries, which contributes to nearly one third of the Jetta hybrid's fuel saving potential. Meanwhile, heating, air conditioning, and electric convenience features such as the audio system continue to operate.</p>
<p>Power is transferred to the front axle via a seven-speed dual clutch (DSG) gearbox similar to Porsche’s popular seven-speed Doppelkupplungsgetriebe (PDK) dual-clutch automated gearbox found in the new <a title="find out more about Porsches 911 Carrera Cabriolet" href="http://blog.truecar.com/2012/01/16/porsche-premieres-911-carrera-cabriolet-at-2012-north-american-international-auto-show/"  target="_blank">911 Carrera</a>, combining driving fun with sustainability.</p>
<p>All-new instrumentation displays battery charge state and energy flow, while a multifunctional display integrated into the left side of the instrument panel replaces the tachometer and informs the driver of the vehicle's operating mode.</p>
<h2><strong><a title="2013-Volkswagen-Jetta-Hybrid-badge" href="http://blog.truecar.com/2012/01/19/volkswagen-reveals-2013-jetta-hybrid-at-2012-north-american-international-auto-show/2013-volkswagen-jetta-hybrid-badge/"  rel="attachment wp-att-19146" target="_blank">
		<span class="pibfi_pinterest">
		<img class="alignright size-full wp-image-19146" style="border-style: initial; border-color: initial; border-image: initial; border-width: 0px; margin: 5px;" title="2013-Volkswagen-Jetta-Hybrid-badge" src="http://tcblog.truecar.com/wp-content/uploads/2012/01/2013-Volkswagen-Jetta-Hybrid-badge1.jpg" alt="2013-Volkswagen-Jetta-Hybrid-badge" width="400" height="266" />
			<span class="xc_pin" onclick="pin_this(event, 'http://pinterest.com/pin/create/button/?url=http://blog.truecar.com/2012/01/19/volkswagen-reveals-2013-jetta-hybrid-at-2012-north-american-international-auto-show/&media=http://tcblog.truecar.com/wp-content/uploads/2012/01/2013-Volkswagen-Jetta-Hybrid-badge1.jpg&description=Volkswagen Reveals 2013 Jetta Hybrid at 2012 North American International Auto Show')">
			</span>
		</span></a>2013 Volkswagen Jetta hybrid Trim and Options</strong></h2>
<p>Three <a title="learn more about trim levels here at TrueCar.com" href="http://blog.truecar.com/2012/01/03/trim-levels-packages-and-options-on-new-cars-%E2%80%93-what-do-they-all-mean-2/" >trims</a> will be offered for the Jetta Hybrid: entry-level S, midrange SE, and top-of-the-line SEL. Standard features include Bluetooth technology, LED taillights, SiriusXM satellite radio, special 15-inch aluminum-alloy wheels, and unique Hybrid body kit and badging. Available premium features will include bi-xenon headlights and LED daytime running lights, a power driver’s seat, heatable front seats, navigation system, rear-view camera, and Volkswagen’s exclusive Fender Premium Audio System. (More details on trim levels will be released closer to the on-sale date.)</p>
<p>Visually the Jetta Hybrid is distinguished by aerodynamic modifications that include a new front airdam, a rear diffuser, and an integrated rear spoiler, a special radiator grille where the VW logo is integrated on a blue background for the first time, and Hybrid signatures on the grille, front fenders, and rear deck.</p>
<p>Standard safety equipment for all Jetta models includes electronic stability control, anti-lock brakes, and six airbags (driver and front passenger airbags, front side thorax airbags, and side curtain airbags front and rear). Volkswagen claims the Jetta Hybrid is the quietest car the manufacturer has ever produced in the compact sedan class, thanks to a newly designed exhaust system, an acoustically optimized windshield, and thicker front side windows.</p>

Then use TrueCar’s Best Local Price feature to find the best target price before you even set foot in the dealership.</p>
